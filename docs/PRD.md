# PRD - 食譜搜尋系統

## 1. 專案概述

### 背景與動機
對於剛開始學做菜的新手廚師來說，面對成千上萬的食譜往往不知從何下手。他們需要一個簡單直覺的工具，能夠快速搜尋到適合自己程度的食譜、上傳自己的料理紀錄、以及收藏喜歡的食譜以供日後參考。「食譜搜尋系統」旨在提供一站式的食譜查詢與管理平台，降低新手下廚的門檻。

### 目標用戶
新手廚師——包含大學生、租屋族、剛開始自炊的上班族等，對烹飪有興趣但經驗不足，希望透過系統快速找到簡單易做的食譜。

### 核心價值主張
提供一個對新手友善的食譜搜尋與管理平台，讓使用者能輕鬆搜尋各種食譜、上傳自己的食譜與他人分享、儲存收藏喜歡的食譜，並透過食材篩選與難度分類，快速找到最適合當下情境的料理。

---

## 2. 功能需求

### 功能一：搜尋食譜
- **使用者故事**：作為新手廚師，我希望能透過關鍵字（食譜名稱、食材、料理類型）搜尋食譜，以便快速找到我想做的料理。
- **功能說明**：
  - 支援以食譜名稱、食材名稱、料理類型等關鍵字進行模糊搜尋
  - 搜尋結果以列表方式呈現，顯示食譜名稱、縮圖、難度、預估時間
  - 可依照難度、烹飪時間、評分等條件進行排序與篩選

### 功能二：上傳食譜
- **使用者故事**：作為新手廚師，我希望能上傳自己嘗試過的食譜（包含照片、食材、步驟），以便記錄自己的料理成果並與他人分享。
- **功能說明**：
  - 提供食譜上傳表單，包含：食譜名稱、食材清單、製作步驟、烹飪時間、難度等級、分類標籤
  - 支援上傳食譜成品照片（圖片檔案上傳）
  - 上傳後食譜即可被其他使用者搜尋到

### 功能三：儲存 / 收藏食譜
- **使用者故事**：作為新手廚師，我希望能將喜歡的食譜收藏起來，以便下次想做菜時可以快速找到。
- **功能說明**：
  - 在食譜詳情頁提供「收藏」按鈕，點擊後將食譜加入個人收藏清單
  - 提供「我的收藏」頁面，集中檢視所有已收藏的食譜
  - 可取消收藏

### 功能四：食譜分類與標籤
- **使用者故事**：作為新手廚師，我希望食譜能有清楚的分類（如：中式、西式、日式、甜點、快速料理等），以便我依照心情或需求瀏覽對應類別的食譜。
- **功能說明**：
  - 系統提供預設分類（如：中式、西式、日式、韓式、甜點、湯品、快速料理等）
  - 上傳食譜時可選擇分類與自訂標籤
  - 首頁或瀏覽頁可依分類篩選食譜

### 功能五：食譜詳情與步驟顯示
- **使用者故事**：作為新手廚師，我希望能看到食譜的完整資訊（食材清單、詳細步驟、烹飪時間、難度），以便按照步驟一步步完成料理。
- **功能說明**：
  - 食譜詳情頁清楚列出食材清單（含份量）
  - 製作步驟以有序編號呈現，每一步附上說明
  - 顯示預估烹飪時間與難度等級
  - 顯示食譜成品照片

### 功能六：編輯與刪除食譜
- **使用者故事**：作為新手廚師，我希望能修改自己上傳的食譜內容，或刪除不再需要的食譜，以便維持食譜資訊的正確性。
- **功能說明**：
  - 使用者可編輯自己上傳的食譜（修改名稱、食材、步驟、照片等）
  - 使用者可刪除自己上傳的食譜
  - 刪除前需確認提示，避免誤刪

### 功能七：根據食材推薦食譜
- **使用者故事**：作為新手廚師，我希望能輸入手邊現有的食材，讓系統推薦可以做的食譜，以便有效利用現有食材、減少浪費。
- **功能說明**：
  - 提供食材輸入欄位，可一次輸入多種食材
  - 系統比對資料庫中的食譜，列出包含這些食材的食譜
  - 依照食材匹配程度排序（完全匹配優先）

## 3. 非功能需求

### 技術限制
- 後端框架：**Flask**（Python）
- 模板引擎：**Jinja2**（Flask 內建）
- 資料庫：**SQLite**（輕量級關聯式資料庫）
- 圖片儲存：存放於伺服器本機的 `uploads/` 資料夾

### 效能考量
- 頁面載入與搜尋回應需流暢迅速
- 搜尋功能使用 SQL LIKE 模糊查詢，資料量在合理範圍內可維持良好效能
- 圖片上傳大小限制為 5MB 以內

### 安全考量
- 使用參數化查詢（parameterized queries）防範 SQL Injection
- 對使用者輸入進行跳脫處理防範 XSS 攻擊
- 上傳檔案需驗證副檔名與 MIME 類型，僅允許圖片格式（jpg、png、gif）
- 本系統為輕量級應用，暫不實作複雜的使用者認證機制（可於未來版本加入）

## 4. MVP 範圍

### Must Have（必要功能）
- 新增 / 上傳食譜（含食材、步驟、基本資訊）
- 瀏覽食譜列表
- 食譜詳情頁（食材清單、步驟、成品照片）
- 關鍵字搜尋食譜
- 編輯與刪除食譜

### Should Have（重要功能）
- 儲存 / 收藏食譜功能
- 食譜分類篩選
- 根據食材推薦食譜

### Nice to Have（加分功能）
- 食譜圖片上傳功能
- 自訂標籤與多重標籤篩選
- 食譜難度與烹飪時間篩選
- 匯入 / 匯出食譜（JSON 格式）
- 使用者註冊與登入系統
